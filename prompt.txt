🚧 1. Project Overview

Project Name: RoadEye

RoadEye is an End-to-End Autonomous Road Infrastructure Monitoring System designed to detect and map road anomalies using Computer Vision.

The system:

- Detects road damage (Cracks & Potholes) using YOLOv8
- Sends predictions through a FastAPI backend
- Stores results and visualizes them on a web dashboard using geospatial mapping
